# מילואומניקים 360

אפליקציית HTML עצמאית למיצוי זכויות, תגמולים ומענקים למשרתי מילואים בישראל (חרבות ברזל והלאה).
עובדת בכל דפדפן ללא שרת — מספיק לפתוח את `index.html`.

## מה יש כאן
- **מחשבון אומדן מענקים 2026** — אינטראקטיבי, רץ 100% במכשיר המשתמש (אין שליחת נתונים).
- **בדיקת זכאות** לפי ימי שמ"פ, מדרג, מצב משפחתי וסטטוס תעסוקתי.
- **כרטיסי מענקים** — תגמול מיוחד, מענק לידה, מעבר דירה, מענק משפחה, ארנק דיגיטלי.
- **קישורים רשמיים בלבד** — צה"ל, ביטוח לאומי, קרן הסיוע.

## מקור-אמת ועדכון נתונים
כל הסכומים והתנאים נמצאים באובייקט `DATA` בראש בלוק ה-`<script>` ב-`index.html`.
המבנה תואם לקובץ `data/grants-2026.json` של הסקיל `miluim-grants-il`.
**לעדכון:** ערוך את `DATA` בלבד + את השדה `last_verified`. אין צורך לגעת בלוגיקה.

> ⚠️ הסכומים הם אומדן. הזכאות הסופית נקבעת ע"י צה"ל / ביטוח לאומי / קרן הסיוע.
> נכון לתאריך העדכון: 28.06.2026.
